# LaTeX Figures: Inkscape and TikZ Workflows

This repository demonstrates two approaches for creating high-quality vector figures in LaTeX documents:

- **`with_inkscape.tex`** — Importing Inkscape SVG figures via the PDF+LaTeX export workflow
- **`with_tikz.tex`** — Drawing a diagram directly in LaTeX using TikZ

---

## Prerequisites

- A LaTeX distribution: [TeX Live](https://tug.org/texlive/) 2018+ or [MiKTeX](https://miktex.org/) 2018+
- [Inkscape](https://inkscape.org/) 1.x (only needed to edit or regenerate the SVG figures)
- LaTeX packages (included in any full TeX Live / MiKTeX installation):
  - `geometry`, `graphicx`, `xcolor`, `bm`, `import` — for `with_inkscape.tex`
  - `tikz`, `amsmath`, `amssymb`, `bm`, `hf-tikz` — for `with_tikz.tex`

---

## Compiling

```bash
pdflatex with_inkscape.tex
pdflatex with_tikz.tex
```

Or with `latexmk`:

```bash
latexmk -pdf with_inkscape.tex
latexmk -pdf with_tikz.tex
```

Pre-compiled PDFs (`with_inkscape.pdf`, `with_tikz.pdf`) are included for reference.

```
    ├── nullspace/          # Least-squares projection geometry
    ├── prisoners/          # Prisoners' problem (100 prisoners)
    └── regression/         # Linear regression scatter plot
```

Each subdirectory under `figures/` contains three files per graphic:
- `name.svg` — editable source file (open in Inkscape)
- `name.pdf` — vector graphic without text (generated by Inkscape)
- `name.pdf_tex` — LaTeX overlay for text positioning (generated by Inkscape)

## Inkscape Export Workflow

To export an SVG for use with this workflow:

1. Open your SVG in Inkscape.
2. Go to **File > Export...** (`Ctrl+Shift+E`).
3. Select **Portable Document Format (*.pdf)** from the format dropdown.
4. Check **Omit text in PDF and create LaTeX file**.
5. Click **Export**.

Inkscape produces two files: `name.pdf` (graphics) and `name.pdf_tex` (text positioning). LaTeX renders both together so that all text is typeset by LaTeX, matching the document font.

## Quick Usage

Add to your preamble:

```latex
\usepackage{graphicx}
\usepackage{xcolor}
\usepackage{import}

\newcommand{\incfig}[3][1]{%
    \def\svgwidth{#1\columnwidth}
    \import{#2}{#3.pdf_tex}
}
```

Include a figure:

```latex
\begin{figure}[ht]
    \centering
    \incfig[0.8]{./figures/myfigure/}{myfigure}
    \caption{Caption text, including any \LaTeX{} math.}
    \label{fig:myfigure}
\end{figure}
```

The optional first argument to `\incfig` controls the figure width as a fraction of `\columnwidth` (default: `1`).
